    title: My first blog

    hexo 构建静态文件无法生成 index.html 等文件

    查看npm安装hexo插件情况

    npm ls –depth 0

    一些插件未安装

    解决

    逐一安装缺失的包

    npm install hexo-generator-archive –save

    npm install hexo-xxx-xxx –save

    …

    ##执行hexo server出现Usage: hexo ”的提示

    出现这个原因是因为没有安装 hexo-server

    #解决

    npm install hexo-server –save
